Univya. Redesign overview · formerly ProfMatch

The Univya redesign — every page, one design language.

Navy primary, teal accent, Plus Jakarta Sans throughout. Each page has a light and dark theme — use the ☾ / ☀ toggle in any header; your choice carries across pages.

Landing page Public

Hero with live search, match preview, problem comparison, how-it-works, FAQ, and final CTA.
Discover Public

Professor directory: search, multi-select filters, result cards with recruiting + confidence signals, empty state.
Dashboard Signed in

Profile summary strip, top matches with fit rings, saved professors with board stages, no-profile empty state.
Matches + intake Signed in

Ranked results with explainable “why matched” evidence, live threshold slider, risk flags, and the 3-step intake wizard.
Pricing Public

Free public discovery vs. Pro personalized workflow, plus the trust-first positioning note.
Auth Public

Sign-in with error state and sign-up, side by side.
Admin suite Internal

All three admin views on one page: dashboard, scan jobs with QA review, and the agentic onboarding wizard.
Components Reference

Nav shells, modals (login gate, report, confirm, outreach draft), toasts, chips, and confidence badges.

Design language at a glance

Navy deep · navy primary · teal accent · soft navy · mist
Type Plus Jakarta Sans 400–800
Radius 10 / 16 / 20 px
Status green=good · amber=caution · red=error, always banded
